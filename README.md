# Memory Matching Game (Tailwind CSS + JavaScript)

A clean, modern, and interactive **Memory Card Matching Game** built using  
**HTML**, **JavaScript**, and **Tailwind CSS**.  
Flip cards, match identical emoji pairs, and try to win using the least number of moves!

---

## How to Play
1. The board contains several face-down cards.  
2. Click any card to flip it.  
3. Click another card to find its matching pair.  
4. If both cards match, they remain flipped.  
5. If they donâ€™t match, they flip back automatically.  
6. Your **Moves** count increases every two flips.  
7. Your **Matches** count increases for every correct pair.

9. Click **Restart** anytime to play again.
